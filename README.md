# 🧮 Math Quiz Game

An interactive and fun **Math Quiz Game** built using **HTML, CSS, and JavaScript**.
Test your math skills with random questions, beat the timer, and climb the leaderboard!

## 🎯 Features

* 🧠 Random Math Questions (Addition, Subtraction, Multiplication)
* ⏱ Timer-based gameplay
* 🎚 Difficulty Levels (Easy / Medium / Hard)
* 🏆 Score Tracking System
* 📊 Leaderboard (Top scores saved using LocalStorage)

## 📸 Preview

> A simple and clean UI with real-time score updates and timer.

## 🛠 Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)
